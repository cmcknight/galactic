To: Prospective Sector Developers
From: Jim V.

The program section2.exe is one I wrote which tries to devide a sector
into its constitutent subsectors so it can be easily incorporated into
Galactic. The program assumes that every world in the sector has a name
and won't function properly if this isn't the case. Aside from outputing
subsector files, if also outputs two additional files, one containing the
base codes that the sector uses (bases) and the other one containing the
allegiance codes which are used (align).

To process the sector spinward.sec, you would type "section2 spinward".
The program assumes the sector file will have a ".sec" extension.
Go ahead and try it at the prompt. spinward.sec is included with this
package. There also a file called al-codes which seems to have been put
together by Leroy Guatney. This may help you with the allegiance codes
definitions. Oh, and last I checked, sector data could be found at:

   http://traveller.is-bremen.de/
   ftp://elendor.sbs.nau.edu/pub/rpg/traveller/sectors

Barring that, I can email you any of the sectors you might need.
